Identifiers

The valid identifier names are described by the following regular expression: [a-zA-Z_]\([a-zA-Z0-9_])*

Examples

  • a, B, c2, d2 valid identifier names.
  • 1, 2a, 3B, 4c5 invalid identifier names.
